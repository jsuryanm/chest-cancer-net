# chest-cancer-net
End-to-End deep learning system for chest cancer detection using Chest CT scans, ML pipelines, and API deployment.


### Workflow of each pipeline components 

1. Update config.yaml
2. Update params.yaml
3. Update the entity 
4. Update the configuration manager in src\cancer_clf\config
5. Update the components
6. Update the  pipeline 
7. Update in main.py
8. Update the dvc.yaml
